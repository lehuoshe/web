0.61 changes
1. Added OEM_LCD4 compile option for AR168M to use dot matrix LCD.
2. Added all keypad related source files into software API.
3. Alex Vangelov ported the tools in sdcc\bin for Linux.
4. Keep PTT low as possbile with AR168R RoIP module.
5. Added OEM_1PORT compile option to use only port 1 of KSZ8842.
6. Added hardware type AR168L for VoIP module with dot matrix LCD.
7. Changed DHCP courtesy behaviour to always retry until successful.
